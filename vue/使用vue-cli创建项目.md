# 使用vue-cli创建项目

## 前提条件

1. 已经安装node并且npm命令可以正常使用

 ## 创建项目

1. 基于webpack的项目

 ```  shell
   vue init webpack orchid-web 
   # orchid-web为项目名
 ```

2. 前期先不要安装eslint 

3. 安装依赖的包

   ``` shell
   npm install
   ```

   

4. 运行项目

   ``` shell 
   npm run dev
   ```
   
5. 打包项目
   
``` shell
    npm run build
```
